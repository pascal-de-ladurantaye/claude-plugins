# Infographic Prompt Examples

## Financial Infographic

### User Request
"I need a visual for our Q3 earnings report"

### Crafted Prompt
```
Create a clean, modern infographic summarizing Q3 2024 earnings for a SaaS company. Include sections for: revenue growth (up 23% YoY), customer acquisition (1,200 new customers), churn rate (reduced to 4.2%), and ARR milestone ($50M). Use a professional dark navy background with white text, accent colors of teal and gold for data highlights. Include minimalist chart visualizations for each metric. Sans-serif typography with clear hierarchy.
```

### Metadata
- **Aspect Ratio**: 4:3
- **Resolution**: 2K
- **Capability Area**: Text Rendering & Infographics
- **Edit Refinements**:
  - "Make the revenue chart more prominent"
  - "Add subtle grid lines to the background"
  - "Increase contrast on the churn rate section"

## Educational Diagram

### User Request
"Explain how photosynthesis works for a biology textbook"

### Crafted Prompt
```
Design an educational diagram explaining the photosynthesis process for a high school biology textbook. Show a cross-section of a leaf with labeled chloroplast, illustrating the light-dependent reactions in the thylakoid and light-independent reactions (Calvin cycle) in the stroma. Use color-coding: blue arrows for water intake, yellow for light energy, green for the chloroplast structures, red for oxygen release, and orange for glucose output. Include chemical equations at each stage. Clean, scientific illustration style with clear labels and directional flow arrows.
```

### Metadata
- **Aspect Ratio**: 16:9
- **Resolution**: 2K
- **Capability Area**: Text Rendering & Infographics
- **Edit Refinements**:
  - "Enlarge the Calvin cycle section for clarity"
  - "Add electron transport chain detail"
  - "Make ATP/NADPH labels more prominent"

## Historical Timeline

### User Request
"Timeline of the space race for a museum exhibit"

### Crafted Prompt
```
Create a dramatic horizontal timeline infographic depicting the Space Race (1957-1975) for a museum exhibit. Start with Sputnik 1 launch, include key milestones: first human in space (Gagarin), first American in orbit (Glenn), Apollo 11 moon landing, and Apollo-Soyuz handshake. Use a deep space background with stars, each event marked by an iconic photograph or illustration. Soviet achievements in red, American in blue. Retro-futuristic typography reminiscent of 1960s NASA mission patches. Include dates and brief descriptions for each milestone.
```

### Metadata
- **Aspect Ratio**: 16:9
- **Resolution**: 4K
- **Capability Area**: Text Rendering & Infographics
- **Edit Refinements**:
  - "Add more detail to the Apollo 11 section"
  - "Include the Luna program achievements"
  - "Adjust star field density in background"

## Process Flow Diagram

### User Request
"Show our customer onboarding process"

### Crafted Prompt
```
Design a professional process flow diagram illustrating a B2B SaaS customer onboarding journey. Five main stages connected by flowing arrows: 1) "Sign Up" (envelope icon), 2) "Welcome Call" (phone icon), 3) "Setup & Integration" (gear icon), 4) "Training Sessions" (graduation cap icon), 5) "Go Live" (rocket icon). Below each stage, show 2-3 sub-steps in smaller text. Use a clean white background with the company's brand colors (specify: coral #FF6B6B and navy #2C3E50). Modern flat design style with subtle shadows. Include estimated timeframes for each stage.
```

### Metadata
- **Aspect Ratio**: 16:9
- **Resolution**: 2K
- **Capability Area**: Text Rendering & Infographics
- **Edit Refinements**:
  - "Add a progress bar element below the stages"
  - "Include customer success manager touchpoints"
  - "Make icons slightly larger"

## Data Dashboard

### User Request
"Marketing metrics dashboard mockup"

### Crafted Prompt
```
Generate a marketing analytics dashboard visualization showing key performance metrics. Layout: top row with 4 KPI cards (Website Traffic: 125K, Conversion Rate: 3.2%, CAC: $45, MQLs: 890). Middle section: large line chart showing 6-month traffic trend with channel breakdown (organic, paid, social, referral). Bottom left: pie chart of traffic sources. Bottom right: bar chart comparing this month vs last month performance. Dark mode interface with a charcoal background (#1a1a2e), neon accent colors for data (cyan #00d9ff, magenta #ff00ff, lime #00ff88). Sleek, modern SaaS dashboard aesthetic.
```

### Metadata
- **Aspect Ratio**: 16:9
- **Resolution**: 2K
- **Capability Area**: Text Rendering & Infographics
- **Edit Refinements**:
  - "Add hover state effect on the traffic trend line"
  - "Include a date range selector in the header"
  - "Add subtle glow effects to the KPI numbers"
